# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Cerdas seriously. If you discover a vulnerability, please follow these steps:

1.  **Do not create a public GitHub issue.**
2.  Email our security team at [security@cerdas.id](mailto:security@cerdas.id) (or the maintainer's email).
3.  Include a detailed description of the vulnerability and steps to reproduce it.

### Response Timeline

-   We will acknowledge your report within 48 hours.
-   We will provide a status update every 5 days.
-   Once fixed, we will credit you in our changelog (unless you wish to remain anonymous).
